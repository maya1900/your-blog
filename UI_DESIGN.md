## 设计语言关键词(来自需求方)

- **高级感** — 留白、克制、精致排版,不堆砌装饰
- **AI 风** — 科技、未来、流动光、渐变、粒子
- **明亮** — 浅色主导,不做深色优先
- **方框** — 卡片化、清晰几何边界、不堆叠圆形 blob

---

## 共同基线(所有方案都遵守)

### 字体
- 西文标题:`Inter` 或 `Geist` (variable font)
- 西文正文:同上
- 中文:`PingFang SC` (mac 系统) / `Noto Sans SC` (web fallback)
- 等宽(代码):`JetBrains Mono`

### 间距尺度
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 (Tailwind 默认延伸)

### 圆角
方框感优先 → 圆角默认 `rounded-lg`(8px)或 `rounded-xl`(12px),不上大圆角

### 阴影
默认很弱(`shadow-sm`),hover 时才显形(`shadow-md`),不滥用

### 动效
- 进入/离开:150–250ms,`ease-out`
- hover:transform + opacity,不做夸张缩放
- 列表卡片 hover:轻微上浮 2px

## 方案 A · 极光 AI 风(Aurora AI)

> **vibe**:像 Vercel、Linear、Cursor 官网的混合体。明亮底色 + 局部极光渐变 + 网格背景 + 锐利方框。

### 色板
```
背景         #FAFAFA   近白,带极轻冷调
卡片         #FFFFFF
主文字       #0A0A0A
次文字       #525252
边框/分割    #E5E5E5
品牌主色     #6366F1   靛蓝
辅色 / 强调  #06B6D4   青蓝
极光渐变     linear-gradient(135deg, #6366F1 0%, #A855F7 50%, #06B6D4 100%)
```

### 关键视觉特征
- 首页 Hero 区域:浅灰底 + 浅网格线背景(`grid-cols-12` 1px 线)
- 文章卡片:纯白底 + 1px 浅灰边框,hover 边框变靛蓝 + 上浮 2px
- 顶部导航:磨砂玻璃(`backdrop-blur`)
- 强调按钮:渐变填充 + 极细高光线
- 标签:小方块徽章,无圆角或仅 `rounded`(4px)
- 后台仪表盘:数据卡片用浅色边框 + 数字加粗到极致(`font-weight: 800`)

### 适合气质
科技博客、AI / 工具类内容、面向技术读者。**最贴合"高级感 + AI 风 + 明亮 + 方框"四个关键词的组合。**

## 方案 B · 杂志极简(Editorial Mono)

> **vibe**:像 Medium、Substack、纽约时报科技版。强排版驱动,几乎没有装饰,信息密度高。

### 色板
```
背景         #FFFFFF
强调底       #F5F5F4   暖灰
主文字       #18181B   近黑
次文字       #71717A
边框         #E4E4E7
点缀色       #DC2626   砖红(仅用于强调/链接 hover)
```

### 关键视觉特征
- 字体优先级压倒一切:标题用 serif(`Fraunces` / `Source Serif`)+ 正文 sans
- 文章列表:**没有卡片**,只有横向分割线 + 大量留白
- 首页 Hero:一段巨大的引文 + 作者名,无图
- 详情页:阅读宽度严格 65–75 字符
- 标签:文字下划线,无背景色
- 后台:表格密集型,黑白灰,操作按钮极简

### 适合气质
写作者博客、长文为主、读者注重阅读体验。**舍弃了 AI 风,换来更纯粹的高级感。**

## 方案 C · 几何方块(Geometric Blocks)

> **vibe**:像 Stripe、Notion 早期、Framer 模板。强方框感、彩色方块拼贴、轻俏的几何装饰。

### 色板
```
背景         #FFFFFF
方块底色     #FEF3C7 / #DBEAFE / #DCFCE7 / #FCE7F3   柔和糖果色
主文字       #0F172A
次文字       #475569
边框(粗)    #0F172A   1.5–2px 实线
品牌主色     #2563EB
```

### 关键视觉特征
- 卡片有明显粗边框(1.5–2px),近似 neo-brutalism 但收敛
- 不同分类用不同糖果底色块标识
- 按钮:实色填充 + 1.5px 边框 + 0 阴影 → hover 时偏移 `translate(2px, 2px)` 露出阴影
- 标签:实色填充小方块
- 首页布局:不对称栅格(Bento 风),不同卡片不同尺寸
- 装饰元素:几何图形(三角、方块、圆点)散点点缀

### 适合气质
个性化博客、设计/创意类内容、年轻活泼但仍保持秩序。**方框感最强,但 AI 风稍弱。**

---

## 方案 D · 玻璃极光(Glass Aurora)

> **vibe**:像 Apple 官网、最新一代 SaaS landing page。明亮底 + 玻璃质感卡片 + 流动渐变光斑。

### 色板
```
背景         #F8FAFC
卡片         rgba(255,255,255,0.7) + backdrop-blur
主文字       #020617
次文字       #64748B
边框         rgba(15, 23, 42, 0.08)
渐变光斑     #818CF8 → #F472B6 → #34D399   (淡化,模糊半径大)
强调色       #0EA5E9
```

### 关键视觉特征
- 首页背景有大半径模糊的彩色光斑(`blur(120px)`),光斑缓慢漂移
- 所有卡片半透明白底 + 模糊 + 极细边
- 数据卡片:数字带渐变文字(`bg-clip-text`)
- 按钮:玻璃质感 + 内发光
- 后台:同语言但光斑更克制
- 方框感来自卡片明确的几何边界,但通过玻璃质感软化锐利

### 适合气质
产品型博客、技术博客、希望带一点"未来感"但仍亲和。**AI 风最足,但需要谨慎控制玻璃元素的密度避免显廉价。**
